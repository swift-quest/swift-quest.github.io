### まとめ

#### 画面に表示

`print` の `()` の中身が表示される。

```swift
print(42)
```

```
42
```

#### 計算

```swift
print(6 + 2) // たし算
print(6 - 2) // ひき算
print(6 * 2) // かけ算
print(6 / 2) // わり算
```

#### 計算順序

```swift
print(1 + 2 * 3)   // かけ算が先 → 7
print((1 + 2) * 3) // たし算が先 → 9
```

`()` を重ねる場合には、算数や数学のように `{}` や `[]` は使わず、内側も外側も `()` を使う。

```swift
print(((1 + 2) * 3 + 4) * 5)
```
